Реализация хранения кэша.
В кэше хранится определённое количество объектов кэша. Если добавляется новый объект кэша, а места для хранения нового объекта недостаточно, то удаляется первый объект кэша, а новый объект становится последним в списке кэша.
Сам объект кэша имеет время жизни, по истечению которого объект уничтожается. Для продления времени жизни кэша, необходимо к нему обратиться. При уничтожении объекта кэша, ресурсы, к которым было обращение, освобождаются.
